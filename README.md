# Practica4

This program is useful to create, consult, remove and modify recipes.

## Install all packages:
```
npm install
```
#### Packages to install:
```

    "graphql-yoga": "^1.18.3",
    "mongodb": "^3.3.5",
    "uuid": "^3.3.3" 
```
#### Execute:
```
npm run dev
```
